# Htaccess generator

You can try it here: https://emirodgar.github.io/htaccess-generator/ (beta)

## Public access options
- Set main domain
- www redirect
- https redirect

## Config options
- Default access page
- Default charset
- Force filetype download
- Limit upload file size
- Set admin mail

## Performance
- Enable Gzip compression
- Enable Keep-Alive headers
- Set Expires Headers

## Custom error pages
- Error 404
- Error 500

## Security options
- Prevent directory navigation
- Disable CGI execution
- Hide server information
- Prevent pages to be framed
- Disable server sign
- Block SPAM bots
- Prevent illegal or unsafe access
- Disallow script execution
- Deny request methods
- Prevent hotlinking
